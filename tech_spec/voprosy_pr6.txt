Практическая работа 6. Возможные вопросы
-----------------------------------------------

В первую очередь, могут быть заданы вопросы на понимание логики работы написанного кода. Например, рассказать в каких местах и какие изменения потребуется внести, чтобы работа программы изменилась необходимым образом.

Другие вопросы

1. Что такое и для чего используется SQL? Приведите примеры распространенных SQL запросов. Какие действия могут выполняться над таблицами? Какие действия могут выполняться над записями таблицы?

2. Каким образом происходит работа с базой данных MySQL из PHP? Расскажите о назначении известных вам функций расширения mysqli.

3. Для добавления заказчика используется следующая форма (ФИО, выбор города из выпадающего списка). Какие таблицы потребуются при реализации?

4. Каким образом в php осуществляется загрузка изображений на сервер и их дальнейшая обработка?

5. Собираемся принимать заявки на товары от постоянных клиентов. Какие таблицы потребуются при реализации. Какие sql-запросы будут использоваться?

6. Какие таблицы потребуются для хранения данных в той или иной ситуации. При реализации:
- рецептах и ингредиентов,
- комментариев к статьям,
- новостной ленты,
- обработки заказов,
- вывода списка похожих товаров и т. п.

Написать запрос, который заполняет эти таблицы тестовыми данными.
Написать запрос, который получает комментарии определенного пользователя за 2019 год с сортировкой по дате.
